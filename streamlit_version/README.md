# Streamlit Version

This folder contains the quick prototype version of Universal Data Analyzer built with Streamlit.

## Features

- Upload CSV or Excel files
- Map dataset columns dynamically
- Calculate CTR, CPC, and ROI
- Show KPI cards and interactive charts
- Filter records by categorical values
- Display simple recommendations based on performance

## Files

- `app.py` - main Streamlit app
- `campaign_data.xlsx` - sample dataset
- `requirements.txt` - Python dependencies for this version

## Run It

```powershell
cd C:\Users\user\OneDrive\Desktop\social_media_tracker
.venv\Scripts\Activate.ps1
pip install -r streamlit_version\requirements.txt
streamlit run streamlit_version\app.py
```
