# Flask Version

This folder contains the more advanced full-stack version of Universal Data Analyzer.

## Features

- User registration and login
- Dataset upload and analysis
- KPI dashboard and interactive charts
- Report saving with SQLite
- PDF and Excel export
- Anomaly detection and simple forecasting

## Files

- `backend/flask_app.py` - Flask backend and analytics logic
- `frontend/index.html` - dashboard page
- `frontend/app.js` - frontend interactions and charts
- `requirements.txt` - Python dependencies for this version

## Run It

```powershell
cd C:\Users\user\OneDrive\Desktop\social_media_tracker
.venv\Scripts\Activate.ps1
pip install -r flask_version\requirements.txt
cd flask_version\backend
python flask_app.py
```

Then open:

```text
http://127.0.0.1:5000
```
