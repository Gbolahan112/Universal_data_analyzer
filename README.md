
# Universal Data Analyzer & Social Media Campaign Tracker

A **universal data analytics web app** built with **Python** and **Streamlit**. This app allows users to upload **CSV or Excel datasets**, analyze key metrics, visualize data, and generate actionable insights. It is ideal for **marketing analytics, business intelligence, and general data exploration**.

---

## 🚀 Features

- Upload any dataset in **CSV** or **Excel** format  
- **Dynamic column mapping** for clicks, impressions, spend, and revenue/ROI  
- Automatically calculates **CTR, CPC, ROI, and performance level**  
- **Key Performance Indicators (KPI)** dashboard  
- Interactive **charts** (bar charts for top performers, ROI vs Spend scatter plots)  
- **Filters** by categorical columns (e.g., age, gender, campaign)  
- **Automated insights and recommendations** based on performance metrics  
- Fully **deployable online** via Streamlit Cloud  

---

## 🎯 How It Works

1. Upload your dataset (`CSV` or `XLSX`).  
2. Map the numeric columns for clicks, impressions, spend, and revenue/ROI.  
3. View automatically generated KPIs, top-performing records, and charts.  
4. Use filters to analyze specific segments of your data.  
5. Read actionable insights generated by the app for your campaigns.

---

## 📊 Example Visualizations

- **KPI Cards:** Avg CTR, Avg ROI, Total Spend  
- **Top Ads/Posts:** Interactive table of top performers  
- **Charts:** CTR performance bar chart, ROI vs Spend scatter plot  
- **Insights:** Automatic recommendations to improve campaigns

---

## 💻 Technologies Used
